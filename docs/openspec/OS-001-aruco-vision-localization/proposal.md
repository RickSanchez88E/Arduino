# OS-001 — ArUco 视觉定位系统

> 状态: 📋 已拆分
> 日期: 2026-03-14
> 提出人: User

## 做什么 (What)

实现基于 OpenCV 和 ArUco Marker 的顶视相机（top-down camera）二维定位系统。在测试场景四周及小车顶部布置预设 ID 的 ArUco 标签，通过 PC 端带线的摄像头实时捕捉画面，识别角点并动态计算出小车在真实世界地面的坐标位置 $(x, y)$ 及航向角 $\theta$。

## 为什么 (Why)

原本计划只依赖电机自带编码器加上纯粹的人工势场法（APF）实现自主寻路，但由于硬件机械误差大、编码器读数及转角误差在复杂运动时不断累积，造成航向和位置严重失真偏移，使得闭环控制失效。为彻底解决因为滑移和摩擦造成的误差累积问题，改用外部观测手段（顶视视觉）进行绝对坐标定位，以便给底层 Arduino 控制板喂养精准的 $(x, y)$ 数据，从而实现后续精准的点到点直线和转向导航控制（规避 APF 先做纯随动）。

## 改哪里 (Where)

初步预估受影响的模块或文件:

## 补充说明

当前的实现只做定位感知（“获取在哪”），暂不包含基于坐标自动控制小车移动的闭环调度动作（“怎么过去”），先把能跑通定位和映射调通。
